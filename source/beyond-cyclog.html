Beyond cyclog

The logging services that are supplied out of the box with the package all use the supplied cyclog tool.

It is not mandatory to use this, and other logging tools exist that can be easily substituted. These include multilog, multilog, svlogd, and s6-log. These tools do more than cyclog, which is aimed at covering the most common use case, but by no means all possible use cases. They can do filtering by pattern, compression at rotation time, general post-processing at rotation time (with saved state between rotations), and single-line "status" files.
